Module eva

Defold Eva library Eva library carry on all basic mobile game functions like iaps, camera, offers and other a lot of basic stuffs.

Designed for fast integration and single code base.

module_settings: { migrations: see migrations_example, window_settings: see window_settings_example festivals_settings: see festival_settings_example, quests_settings: see quest_settings_example, }
